## System Overview

This project is a centralized event discovery platform for CPUT students. Many events across campuses are often missed because information is scattered across posters, social media, and private groups.  

The system provides a single platform where students can view and register for events, while organizers can create and manage them. It aims to make event information more accessible and improve student participation across campus.
